## 关系选择器
子选择器 vs 后代选择器

子选择器 `>` 只作用于其第一代元素 直接子元素，比如子子元素无效。
body > span

后代选择器作用于N代元素。
body span

兄弟选择器 `~`

相邻兄弟选择器 `+`

## 布局方向

块布局，垂直布局
